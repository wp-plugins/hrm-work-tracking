New HRM work tracking plug ins is a complete employee time tracking in the WordPress backend. The working time recording.

== Description ==

Do you work primarily with WordPress? Do you have several employees who are engaged in the blog? Do you have developers who anyway have to log into WordPress, because they work with them? Did you ever a time recording of working hours?
New HRM work tracking plug ins is a complete employee time tracking in the WordPress backend. The working time recording also works for WordPress users that are classified only as a Subscriber.

The plugin is simple: the user logs in and starts his working hours. However, the system has property to capture this time also when he logs out or computers / browser crashes.

Users can capture
- the breaks over the surface,
- knock,
- sign ill.

The security measures of the plug-in the is also impressive. The user activated the plugin, is the only one  in the backend may edit and also the only one who sees the additional staff lists. Should he not be the only one, he can set additional user IDs which also belong to the human resources department.

Everyone in the HR Department can change employee profiles and set the desired hours of the employees, even if this WordPress is user not as an administrator.

The plugin is currently in German, English and Polish, also .po files will be supplied to transfer to other languages.

Installation Note: After activating the plugin, once log out and log back in. You must pull the dashboard widget with time and attendance even in the field of view. There is also still a little widget that the hiring Manager displays the boss. This little widget displays also the phone number if it is stored in the metadata at "tel".

== Installation ==

1. Upload 'hrm-work-tracking' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Log out and Log in again and it starts tracking your time.
